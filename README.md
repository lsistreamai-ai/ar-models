# AR Models

GLB models for AR Marker Manager.

## How to Use

1. Upload your .glb files to this repo
2. Get raw URL: `https://raw.githubusercontent.com/lsistreamai-ai/ar-models/main/YOUR-MODEL.glb`
3. Use that URL in AR Marker Manager
